How we work

  1. Define the goal

    We talk about what you need. What's the goal? Who's the audience?

  2. Prototype the idea

    We come up with 3 concepts. Different approaches, same goal. You pick the one you like.

  3. Build the experience

    We design and build it.

  4. Launch

    You get the finished project. We can set up analytics and run campaigns too, if you want.

  5. Improve

    After launch, we look at real data and make it better.
